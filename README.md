## ⚡ What is this?

A **single HTML file** that turns your raw findings into a polished, professional security vulnerability report complete with live preview, PDF export, and Markdown copy. Built for bug bounty hunters, pentesters, and security researchers who are tired of formatting reports from scratch.

---

## 🔥 Features at a Glance

| | Feature | Details |
|---|---------|---------|
| 🧾 | **Structured Form** | Title, severity, CWE, CVSS, endpoint, parameters, HTTP traffic, code snippets |
| 🎨 | **Live Preview** | See your report render in real time as you type |
| 🚦 | **Severity Badges** | Critical / High / Medium / Low / Info — color-coded automatically |
| 🔢 | **CVSS Auto-Rating** | Type a score → rating fills itself in |
| 💻 | **Code Blocks** | HTTP, Payload, Bash, Python, JS, JSON, XML and more |
| ✍️ | **Inline Formatting** | Select text → tooltip → wrap as `code`, **bold**, or a full code block |
| ➕ | **Custom Sections** | Add unlimited sections with your own titles |
| ⠿ | **Drag to Reorder** | Rearrange report sections by dragging |
| 📋 | **Copy as Markdown** | One click → GitHub-flavored Markdown, ready to paste anywhere |
| 🖨️ | **PDF Export** | Print-optimized layout, no page-break glitches |
| 💾 | **Autosave** | Survives page reloads — localStorage keeps your work safe |
| 🎨 | **6 Accent Themes** | Red · Blue · Green · Violet · Teal · Ash |
