# Pwn Paper
Pwn Paper is a simple web tool that helps security researchers generate structured bug bounty reports quickly.

🔗 Live: https://spondonsaha.github.io/pwn-paper/

## Features
* Add report details (title, summary, steps, impact, remediation)
* Drag and reorder sections
* Generate a clean PDF report
* No login required


## Why to use pwn-paper?
* Writing bug reports takes time and is often repetitive.
* Pwn Paper makes it faster and easier so researchers can focus more on finding bugs.
